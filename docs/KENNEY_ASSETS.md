# Kenney.nl 游戏素材资源清单

本文档记录了 Kenney.nl 上可用于「小青蛙的奇妙冒险」游戏的免费素材资源。

## 概述

Kenney.nl 提供大量 CC0 许可证的游戏素材，可免费用于商业项目。以下是筛选出的可用资源包。

---

## 1. Platformer Pack New（新平台跳跃包）

- **资源数量**: 440+ 个素材
- **包含内容**:
  - 角色精灵图（多种动作状态）
  - 地形块（草地、石头、水面等）
  - 装饰物（树木、岩石、花朵等）
  - UI 元素（按钮、面板、图标）
- **适用场景**: 扩展游戏关卡、增加视觉多样性
- **链接**: https://kenney.nl/assets/platformer-pack-new

---

## 2. Mobile Controls（移动端控件）

- **资源数量**: 900+ 个素材
- **包含内容**:
  - 虚拟摇杆
  - 方向键按钮
  - 动作按钮（跳跃、攻击等）
  - 触控手势提示
- **适用场景**: 微信小游戏移动端控制优化
- **链接**: https://kenney.nl/assets/mobile-controls

---

## 3. Input Prompts（输入提示图标）

- **资源数量**: 1280+ 个素材
- **包含内容**:
  - 键盘按键图标
  - 手柄按钮图标
  - 鼠标/触控提示
  - 多种配色方案
- **适用场景**: 教程关卡、操作提示
- **链接**: https://kenney.nl/assets/input-prompts

---

## 4. Retro Textures（复古纹理）

- **资源数量**: 115+ 个素材
- **包含内容**:
  - 像素风背景纹理
  - 地板/墙面材质
  - 环境特效纹理
- **适用场景**: 关卡背景装饰、氛围营造
- **链接**: https://kenney.nl/assets/retro-textures

---

## 使用建议

1. **优先使用 Platformer Pack** - 与当前游戏风格最接近
2. **Mobile Controls** - 如需改进微信小游戏触控体验
3. **Input Prompts** - 制作新手引导时使用
4. **Retro Textures** - 丰富背景层次感

## 许可证

所有素材均为 CC0 (Creative Commons Zero) 许可证：
- 可用于商业项目
- 无需署名
- 可自由修改和分发
