# 玩家角色图片规格

## 基本信息

- **角色名称**：茶叶蛋
- **原始尺寸**：40×50 像素
- **推荐制作尺寸**：80×100 或 160×200 像素
- **格式**：PNG（透明背景）

## 图片清单

| 文件名 | 动作状态 | 设计要点 |
|--------|---------|---------|
| `idle_1.png` | 原地呼吸(帧1) | 正常站姿，轻微上下浮动 |
| `idle_2.png` | 原地呼吸(帧2) | 比帧1低2-3px，眼睛可微闭 |
| `walk_left_1.png` | 左走(帧1) | 身体左倾约5°，左脚前 |
| `walk_left_2.png` | 左走(帧2) | 身体左倾约5°，右脚前 |
| `walk_right_1.png` | 右走(帧1) | 身体右倾约5°，左脚前 |
| `walk_right_2.png` | 右走(帧2) | 身体右倾约5°，右脚前 |
| `jump.png` | 跳起 | 身体收缩，表情惊讶，双脚离地 |
| `fall.png` | 下落 | 身体伸展，衣服/裂纹向上飘 |

## 角色设计参考

- **主体颜色**：#F5E6D3（蛋壳色）
- **裂纹颜色**：#8B7355（棕色）
- **眼睛**：黑色圆点 + 白色高光
- **嘴巴**：简单的W形或直线

## 注意事项

1. 所有图片必须尺寸一致
2. 背景透明
3. 角色居中，周围留 2-3px 边距避免裁切
4. 左右行走可以互为镜像
